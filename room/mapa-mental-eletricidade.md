# Eletricidade

## Carga Elétrica e Estrutura da Matéria
- Átomo
  - Próton (+) — núcleo
  - Nêutron (0) — núcleo
  - Elétron (-) — eletrosfera
- Corpo neutro
  - Nº prótons = Nº elétrons
- Corpo eletrizado positivamente
  - Perde elétrons
- Corpo eletrizado negativamente
  - Ganha elétrons
- Princípios
  - Cargas iguais se repelem (+,+) (-,-)
  - Cargas diferentes se atraem (+,-)
- Conservação da carga
  - Não é criada nem destruída
  - Apenas transferida

## Condutores e Isoladores
- Condutores
  - Possuem elétrons livres
  - Conduzem eletricidade
  - Ex: ouro, prata, cobre, alumínio, ferro, água com sais
- Isoladores
  - Elétrons presos aos átomos
  - Não conduzem facilmente
  - Ex: borracha, madeira seca, plástico, vidro, porcelana, papel

## Lei de Coulomb
- Fórmula: F = k·|q1·q2| / d²
- k (vácuo) = 9×10⁹ N·m²/C²
- Aumenta quando
  - as cargas aumentam
- Diminui quando
  - a distância aumenta
- Distância ao quadrado
  - Dobra a distância → F fica 1/4
  - Triplica a distância → F fica 1/9
- Exemplo resolvido
  - q1=2C, q2=3C, d=2m
  - F = 13,5×10⁹ N

## Campo Elétrico
- Fórmula: E = F/q
- Fórmula: E = k·Q/d²
- Unidade: N/C (Newton por Coulomb)
- Direção do campo
  - Carga positiva → aponta para fora
  - Carga negativa → aponta para dentro

## Força Elétrica
- Fórmula: F = q·E
- Sentido da força
  - Carga positiva → acompanha o campo
  - Carga negativa → oposta ao campo

## Linhas de Força
- Representam graficamente o campo elétrico
- Indicam
  - Direção
  - Sentido
  - Intensidade
- Regras
  - Saem das cargas positivas
  - Chegam nas cargas negativas
  - Nunca se cruzam
  - Mais próximas = campo mais intenso
  - Mais afastadas = campo mais fraco
- Configurações
  - Positiva e negativa → linhas vão do + para o -
  - Duas positivas → linhas se afastam (repulsão)
  - Duas negativas → linhas chegam nas cargas

## Eletrização
- Por atrito
  - Dois corpos esfregados
  - Um perde, outro ganha elétrons
  - Ex: vidro + seda
- Por contato
  - Corpo eletrizado toca outro
  - Cargas se distribuem
  - Ficam com cargas de mesmo sinal
- Por indução
  - Sem contato, só aproximação
  - Campo elétrico separa as cargas

## Resumo das Fórmulas
- F = k·|q1·q2| / d² (Coulomb)
- E = F/q (Campo elétrico)
- E = k·Q/d² (Campo por uma carga)
- F = q·E (Força elétrica)

## Dicas para a Prova
- Memorizar as 3 fórmulas principais
- Cargas iguais → repulsão
- Cargas diferentes → atração
- Campo sai do positivo, chega ao negativo
- Linhas nunca se cruzam
- Maior distância → menor força
- Condutores têm elétrons livres
- Isoladores dificultam a corrente
